# Outlook and Teams Troubleshooting Guide

## Purpose

This guide helps employees resolve common Outlook and Microsoft Teams issues.

## Outlook Issues

Common symptoms:
- Outlook does not open.
- Emails are not syncing.
- Calendar invites are missing.
- Search is not working.
- User cannot send or receive email.

Basic troubleshooting:
1. Restart Outlook.
2. Check internet connection.
3. Use Outlook Web Access to confirm mailbox access.
4. Recreate Outlook profile if sync problems continue.
5. Check mailbox storage limits.
6. Restart the laptop.

## Teams Issues

Common symptoms:
- Teams does not open.
- User cannot join meetings.
- Audio or camera does not work.
- Messages are delayed.
- Teams sign-in fails.

Basic troubleshooting:
1. Restart Teams.
2. Check microphone and camera permissions.
3. Sign out and sign back in.
4. Try Teams web version.
5. Clear Teams cache if issues continue.
6. Restart the laptop.

## Escalation

Create a ticket if:
- Outlook or Teams fails after restart and web access test.
- Sign-in fails repeatedly.
- Calendar or mailbox data appears missing.
- Audio/camera issue affects meetings after permissions are checked.
