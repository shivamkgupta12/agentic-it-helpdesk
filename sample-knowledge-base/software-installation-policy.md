# Software Installation Policy

## Purpose

This policy explains how employees can request software installation.

## Approved Software

Some software may be pre-approved by IT. Examples:
- Microsoft Teams
- Microsoft Outlook
- Company VPN client
- Approved browsers
- Standard productivity tools

## Software Requiring Approval

The following usually require approval:
- Design tools such as Figma.
- Developer tools.
- Paid software.
- Software requiring admin privileges.
- Software that stores company data externally.
- Browser extensions that access sensitive data.

## Required Information

For software installation requests, collect:
1. Software name.
2. Business purpose.
3. Employee device type.
4. Operating system.
5. Whether a paid licence is required.
6. Manager approval if required.

## Figma Requests

Figma may require approval because it can store design files and company information externally. The request should include business justification and manager approval where required.

## Ticket Handling

Create a support request or incident with:
- Software name.
- Business justification.
- Device information.
- Requested installation date.
- Approval requirement.

## Suggested Ticket Category

Category: Software Request  
Priority: Low  
Urgency: Medium
