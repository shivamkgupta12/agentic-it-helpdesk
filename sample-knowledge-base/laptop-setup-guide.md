# Laptop Setup Guide

## Purpose

This guide covers common laptop setup and device onboarding steps.

## Standard Setup Checklist

1. Confirm the employee can log in.
2. Connect to Wi-Fi.
3. Install or verify VPN client.
4. Install Microsoft Teams and Outlook.
5. Confirm endpoint protection is active.
6. Confirm disk encryption is enabled.
7. Apply operating system updates.
8. Confirm access to required internal systems.

## Common Issues

Employees may report:
- Cannot log in to laptop.
- Laptop is slow after setup.
- VPN client missing.
- Outlook profile not configured.
- Teams not signing in.
- Device compliance warning.

## Escalation

Create a ticket if:
- Device compliance checks fail.
- User cannot log in after password verification.
- Required applications are missing.
- Endpoint protection is disabled.
- The device is lost or stolen.
